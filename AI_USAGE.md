# AI Usage

This project was developed with assistance from AI tools while ensuring that all generated code was reviewed, understood, tested, and modified before integration.

* ChatGPT
* Google Gemini Documentation
* MDN Web Docs

## AI Assisted In

* JWT Authentication
* Refresh Token implementation

* PDF parsing guidance
* Debugging
* Error handling
* Project architecture
* API design
* Documentation

## Developer Responsibility

All AI-generated suggestions were carefully reviewed, tested, and modified where necessary before being added to the project. Final implementation decisions, debugging, integration, and validation were completed manually.
